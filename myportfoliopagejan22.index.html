RFID lock solution

By reverse engineering the electronic RFID locks being used for patient bed side lockers, I was able to create a bespoke RFID programming and database system. This system of programming allows us to have a more robust and reliable system by reducing regular breakdowns and improving patient safety. This project has allowed me to develop a detailed understanding of passive and active RFID technology.
